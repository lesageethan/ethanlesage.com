Rust BaseFinder

View the site here.

Rust BaseFinder is a site made in 2020, which serves as a forum to share, compare, and review base designs for a video game called Rust. Since its inception it has received over 1 million pageviews, 200 thousand users, and 2 thousand signups.

The site was built from the ground-up by me, using Django for the backend, SQLite for the database, and HTML, CSS and Bootstrap for the frontend. As a result, the site is relatively lightweight (before Google covers it in Adsense and Analytics scripts), and can be ran on a bottom-of-the-range DigitalOcean droplet.
